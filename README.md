# 💈 Navalha - Sistema de Gerenciamento



Sistema web desenvolvido para gerenciamento de uma barbearia, com foco em agendamento de serviços, controle de clientes e organização da rotina de atendimentos.

## 📌 Sobre o Projeto

O **Navalha** é uma solução digital criada para substituir processos manuais (como anotações em papel ou mensagens informais), proporcionando:

- Maior organização
- Redução de erros
- Melhor experiência para clientes e barbeiros

O sistema permite que clientes agendem horários facilmente e que o barbeiro gerencie sua agenda de forma eficiente.

---

## 🎯 Objetivos

- Facilitar o agendamento de serviços
- Organizar a agenda da barbearia
- Centralizar o controle de clientes
- Reduzir conflitos de horários
- Melhorar a experiência do usuário

---

## 👤 Público-Alvo

- Barbearias de pequeno e médio porte
- Profissionais autônomos da área de estética
- Negócios que utilizam agendamento de serviços

---


## 🧠 Metodologia

O projeto foi desenvolvido utilizando a metodologia ágil **Kanban**, com organização via Trello, permitindo:

- Visualização do fluxo de tarefas
- Priorização de demandas
- Acompanhamento do progresso em tempo real
